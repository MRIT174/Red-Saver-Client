# RedSaver

Live URL: [https://red-saver-blood.web.app/](https://red-saver-blood.web.app/)

## Project Purpose
RedSaver is a web application designed to help connect blood donors with those in need of blood donations. The platform facilitates blood donation requests, tracks donations, and enables secure fund collection for medical emergencies.

## Key Features
- User Registration & Authentication: Users can register, login, and manage their profiles. Authentication is handled via JWT and Firebase tokens.
- Blood Donation Requests: Users can post and manage blood donation requests.
- Fund Management: Donors can contribute funds securely using Stripe integration.
- Admin Panel: Admins can manage users, block/unblock users, and update user roles.
- Division & District Support: Easily select and filter by geographical divisions and districts.
- Real-time Updates: Users can see the latest donation requests and funds in real-time.
